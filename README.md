# Victoria Billington

- Victoria Billington Portfolio Site
- Built with Vuejs, Sass
- Uses Contentful.io for CMS
- Hosted on Netlify
- Auto deploy on commit from Github.com
- Was a great project for learning SASS / CSS

## Install

npm install

### START

npm start

### BUILD

npm run build
